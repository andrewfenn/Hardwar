Hardwar

An open source remake of the classic game Hardwar

Download .zip Download .tar.gz View on GitHub

Goals

The goal of this project is to produce an open source remake of the classic game Hardwar that will be operating system independent. The aim is to create something new and original while staying true to the spirit of the original game.

Community

A lot of the communication happens on the forum so feel free to stop by.

This isn't the original Hardwar game

This is a re-make of another game with the same name which is also called "Hardwar". The game is currently is pre-alpha state. There is currently no game to play.

Development

If you are interested in helping develop the game then please see below for build instructions.

Build prerequisites

Hardwar requires the following libraries be pre-installed.

  • Ogre3D - 1.8.0
  • OIS - 1.2
  • CMake
  • The Boost libraries
  • CEGUI

Building

The building instructions should be similar regardless of if you're on windows, osx or linux.

Linux

mkdir hardwar (git clone here)
mkdir build && cd ./build
cmake ../hardwar
make

Windows and OSX

Copy the same steps as above, create your git repo and use cmake to generate either visual studio project files or xcode files for your platform.

You should then only have to open the xcode or visual studio file and compile. If that doesn't work please file bugs or pull requests.
